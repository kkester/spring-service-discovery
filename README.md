# Spring Service Discovery

### Overview

This PoC demonstrates the following architecture, design, and coding strategies:

1. Eureka Client and Server.

### Running the Application

1. Run `EurekaClientApplication`
1. Run `EurekaServerApplication`
1. View the [Eureka Dashboard](http://localhost:8761/)
